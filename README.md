### Getting started

Choose your own names. I'll use placeholders!


In the command line:
* `mkdir new-project`
* `cd new-project`
* `npm init -y`
* `npm i jest --save-dev`
* `atom .`

In package.json, inside scripts
* `"test": “jest —watchAll"`

In the command line:
* `npm run test`

Then write your tests and code!
